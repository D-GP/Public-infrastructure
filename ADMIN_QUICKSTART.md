# 🚀 Admin Portal - Quick Start Guide

## Welcome to Your Admin Portal!

This guide will help you get started with managing complaints for your department.

---

## 🔐 Login (First Time)

### Step 1: Go to Admin Portal
```
🌐 Open your browser and go to:
http://your-server-address/admin/login
```

### Step 2: Enter Your Credentials
```
📧 Email: Enter your official email address
🔒 OTP Method: Choose "Email" (SMS coming soon)
```

### Step 3: Receive OTP
```
✉️ Check your email inbox
🔑 Look for a message with subject: "Your Admin Portal Login OTP"
📝 Copy the 6-character code shown
```

### Step 4: Verify OTP
```
1. Go back to the portal (otp-verify page should appear)
2. Paste the 6-character code
3. Click "Verify & Login"
⏰ The code expires in 10 minutes - hurry!
```

### Step 5: Access Dashboard
```
✅ You're now logged in!
📊 You'll see your department dashboard
```

## 📊 Dashboard Overview

### Your Dashboard shows:

**Key Metrics** (Top cards):
- 📋 **Total Complaints**: All complaints in your department
- ⏳ **Pending**: Awaiting action
- ⚙️ **In Progress**: Being worked on
- ✅ **Resolved**: Successfully completed
- 🚨 **High Priority**: Urgent unresolved cases
- ⏱️ **Avg Resolution**: Average days to resolve

**Charts** (Middle section):
- 📈 **Status Breakdown**: Visual breakdown of complaint statuses
- 🎯 **Priority Distribution**: How complaints are distributed by priority

**Recent Complaints** (Bottom):
- Shows last 5 complaints
- Click "View All →" to see complete list
- Quick action buttons

### Dashboard Actions:
```
🔍 Search: Type to find complaints
🔄 Refresh: Get latest data
🔔 Notifications: View alerts (badge shows count)
```

## 📋 Managing Complaints

### View All Complaints
```
1. Click "Complaints" in left sidebar
2. See all complaints for your department
3. Complaints are sorted newest first
```

### Search for a Complaint
```
🔍 Type in the search box:
   - Complaint title
   - Reporter name
   - Location
   - Description keywords
Results update in real-time!
```

### Filter Complaints
```
By Status:
  📌 All Status
  ⏳ Pending
  ⚙️ In Progress
  ✅ Resolved
  ❌ Rejected

By Priority:
  🚨 High
  ⚠️ Medium
  ℹ️ Low

Sort By:
  📅 Newest First
  📅 Oldest First
  🚨 Highest Priority
```

### View Complaint Details
```
1. Find the complaint in the list
2. Click "View Details" button
3. A modal window opens showing:
   - Full description
   - Reporter information
   - Location
   - Date created
   - All attached images
   - Current status
```

### See Attached Images
```
1. Open complaint details
2. Scroll to "Attached Images" section
3. Click on any image to expand/view full size
4. Images are typically:
   - Problem photographs
   - Location proof
   - Evidence documentation
```

### Update Complaint Status
```
1. Open complaint details
2. Scroll to "Update Status" section
3. Select new status from dropdown:
   - Pending: Just received
   - In Progress: Currently working on it
   - Resolved: Problem fixed
   - Rejected: Cannot address
4. Optional: Add notes explaining the change
5. Click "Update Status"
6. Reporter automatically receives email notification!
```

## 💡 Best Practices

### Status Updates
✅ **Do:**
- Update status regularly
- Add explanatory notes
- Close complaints as soon as resolved
- Keep descriptions accurate

❌ **Don't:**
- Leave complaints stuck in one status
- Use vague notes
- Forget to notify stakeholders
- Ignore high-priority items

### Response Time
- 🎯 Target: Respond within 24 hours
- 🚨 High Priority: Within 12 hours
- 📊 System tracks your average response time

### Organization
- 📌 Group similar complaints
- 🔄 Work on one category at a time
- 📋 Use filters effectively
- 🔍 Search to avoid duplicates

## 🆘 Troubleshooting

### Didn't Receive OTP Email?
```
1. Check spam/junk folder
2. Wait a few seconds
3. Click "Resend OTP"
4. Check if email is correct
5. Contact admin support if persistent
```

### Forgot Your Password?
```
1. Go to login page
2. Click "Contact Support"
3. Provide your email
4. Receive password reset link
(Coming soon - for now, contact your admin)
```

### Can't See Other Department Complaints?
```
✅ This is normal!
Each admin only sees their department's complaints.
This protects data privacy and keeps it organized.
```

### Charts Not Showing?
```
1. Refresh the page (F5)
2. Clear browser cache
3. Try a different browser
4. Check if JavaScript is enabled
```

### Getting Logged Out?
```
Your session expires after 1 hour for security.
Just login again with OTP - quick and easy!
```

## 🔒 Security Tips

### Protect Your Account
```
🔐 DO:
- Use strong unique password
- Never share your OTP code
- Logout on shared computers
- Update password regularly

❌ DON'T:
- Write down OTP codes
- Leave browser open unattended
- Share login credentials
- Use on public WiFi without VPN
```

### Data Privacy
```
✅ All data is encrypted
✅ Only your department's data visible
✅ HTTPS connection (secure)
✅ Automatic logout for safety
```
